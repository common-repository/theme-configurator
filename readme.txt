== Description ==

theme-configurator is the easiest and most elegant way to add customized options pages to your Wordpress theme. No coding required!

Watch a [video tutorial](http://www.l90r.com/posts/theme-configurator) or read its [transcript](http://www.l90r.com/posts/theme-configurator)

== Installation ==

Download and activate the plugin through the 'Plugins' menu in WordPress, just like any other WordPress plugin. Customizing the options pages is easy and graphical. Watch/read a tutorial [here](http://www.l90r.com/posts/theme-configurator)
